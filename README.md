# Care Toys - Premium E-Commerce Experience

An immersive, high-performance e-commerce platform for premium toys and collectibles, built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

-   **Premium Design**: Anti-gravity racing theme with dark mode aesthetics.
-   **Dynamic Animations**: Powered by Framer Motion and GSAP.
-   **E-Commerce Functionality**: Product browsing, cart management, and checkout flow.
-   **Responsive**: Optimized for all devices from mobile to desktop.
-   **Performance**: Fast page loads with Next.js App Router and server components.

## Tech Stack

-   **Framework**: Next.js 14 (App Router)
-   **Language**: TypeScript
-   **Styling**: Tailwind CSS, ShadCN UI
-   **Animation**: Framer Motion
-   **Icons**: Lucide React

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/sheikmohaidee/care-toys.git
    cd care-toys
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
